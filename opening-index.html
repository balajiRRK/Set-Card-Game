Instructions

  • The goal of the game is to find all possible sets of three cards where each card's features are either all the same or all different.
  • A set consists of three cards in which each of the cards' features (number, shape, shading, and color) are either all the same or all different.
  • To select a set, press the corresponding keys for the three cards and confirm your selection by pressing "Enter".
